# Open in Chrome Skill

This skill provides the capability to launch the local `index.html` file in Google Chrome on macOS.

## Command

```bash
open -a "Google Chrome" hacker-terminal/index.html
```

## Usage Guidelines

1. Use this command whenever the user wants to see the visual output of the `hacker-terminal` project.
2. Ensure you are in the root directory of the `Fallout-GN` project before running.
